# NextAutoSell

Venda os seus minérios automaticamente ao quebrá-los.

## Commands

| Comando    | Descrição | Permissão |
|------------|-------------------|------------------------------------------------|
|/nas        | Mostra a mensagem de ajuda | `nextautosell.command`                          |
|/nas reload | Recarrega os arquivos de configurações | `nextautosell.command.reload`    |

## Download

Você pode encontrar a .JAR para baixar [clicando aqui](https://github.com/gabrielbs21/oreautosell/releases).

## Dependencies

A única dependência deste plugin é o [Vault](https://dev.bukkit.org/projects/vault) e um plugin de economia (recomendamos utilizar o [NextEconomy](https://github.com/NextPlugins/NextEconomy)).
